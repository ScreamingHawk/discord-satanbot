# SatanBot

A bot for The 7th Circle discord server

## Usage

### Install

Install `node` and `yarn`

Install dependencies

```
yarn
```

Copy the example `.env` file and update the contents with the bot's token.

```
cp .env.example .env
```

### Run

Run the bot

```
yarn start
```
